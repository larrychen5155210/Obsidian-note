### 🌐 網頁服務列舉 (Web Enumeration)
*對 80/443 等網頁連接埠進行深入探測。*

#### 📁 目錄爆破 (Feroxbuster / Gobuster)
```bash
feroxbuster -u http://192.168.x.x/ -w /usr/share/wordlists/dirb/common.txt -x php,txt,html -t 50 -d 2
```
**發現的敏感路徑：**
- `http://192.168.x.x/robots.txt` - *備註說明*
- `http://192.168.x.x/admin/` - *後台登入頁面*

### 📂 其他服務列舉 (FTP / SMB / SNMP 等)
> [!tip] **服務探測紀錄**
> *   **FTP**：嘗試匿名登入 (`anonymous` / `anonymous`)
> *   **SMB**：使用 `enum4linux` 或 `nxc smb` 檢查是否有 Guest 共享目錄

```bash
# 範例指令
nxc smb 192.168.x.x -u '' -p '' --shares
```
