# {{title}} 簡介

本筆記摘要自詳細報告 [[{{title}}-Walkthrough]]。以下是該靶機的快速滲透路徑：

## 1. 偵察與列舉 (Reconnaissance)
*   **連接埠掃描**：開啟連接埠 `22 (SSH)` 與 `80 (HTTP)`。
*   **服務與目錄列舉**：對特定服務進行爆破或分析，例如在 `/robots.txt` 發現敏感路徑。

## 2. 初始存取 (Initial Access)
*   **漏洞利用 / 憑證獲取**：透過弱點利用或設定檔分析，取得管理員帳密。
*   **Shell 取得**：執行逆向 Shell (Reverse Shell) 或透過 SSH 登入，取得初始系統存取權（`local.txt`）。

## 3. 特權提升 (Privilege Escalation)
*   **權限提升向量**：經由本地列舉（如 SUID/SGID 檔案、可寫定時任務或 Sudo 權限限制不當）發現可提權點。
*   **取得 Root / SYSTEM**：使用對應的提權腳本或指令取得最高權限（`proof.txt`）。
